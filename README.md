# 📊 Customer Churn Analysis

## 📌 Project Overview
This project analyzes customer churn in a telecom company to identify key factors affecting customer retention.

## 🎯 Objectives
- Analyze customer data
- Identify churn patterns
- Provide business insights

## 🧹 Data Cleaning
- Handled missing values
- Converted data types
- Removed duplicates

## 📊 Exploratory Data Analysis
- Distribution of numerical features
- Churn analysis
- Monthly charges impact

## 📈 Key Insights
- High-paying customers tend to churn more
- Month-to-month contracts have highest churn
- Long-term contracts improve retention
- Electronic check users churn more

## 👥 Customer Segmentation
- Customers divided into 3 groups based on tenure
- Long-term customers generate higher revenue

## 🧠 Technologies Used
- Python
- Pandas
- Matplotlib
- Jupyter Notebook

## 📂 Project Structure
- `data/` → Dataset
- `notebook/` → Jupyter Notebook
- `report/` → Final Report
- `screenshots/` → Graph images



## 🚀 Conclusion
Customer churn is influenced by pricing, contract type, and payment method. Companies should focus on long-term customer retention strategies.
